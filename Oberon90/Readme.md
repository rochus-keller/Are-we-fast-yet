This is a version of the "Are We Fast Yet?" benchmark suite migrated to the Oberon 90 language.

The implementation was directly derived from the C implementation from https://github.com/rochus-keller/Are-we-fast-yet/tree/main/C
by Rochus Keller supported by Perplexity.

All benchmarks have been migrated and work with the [AoCodeNavigator2](https://github.com/rochus-keller/activeoberon/) (when
transpiled to C99 and compiled with GCC). 

The subset Richards, Bounce, List, Mandelbrot, Permute, Queens, Sieve, and Storage was successfully run on Oberon System V4 x86.
See the Results directory for the V4 performance measurement results.
