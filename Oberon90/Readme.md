This is a version of the "Are We Fast Yet?" benchmark suite migrated to the Oberon 90 language.

The implementation was directly derived from the C implementation from https://github.com/rochus-keller/Are-we-fast-yet/tree/main/C
by Rochus Keller supported by Perplexity.

NOTE: this is work in progress, not all benchmarks are migrated yet (Json, CD and Deltablue are missing)

The code is currently compatible with Oberon System V4 release 1.7.02 and Oberon System 3 release 2.3.7.

Currently, the following benchmarks are known to work: Richards, Bounce, List, Mandelbrot, Permute, Queens, Sieve, Storage,
NBody, Towers and Havlak.

See the Results directory for V4 performance measurement results (with a subset of the benchmarks available and working at the time).
